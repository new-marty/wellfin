## Roadmap

### MVP

- SharedKit DTOs (User, Account, Transaction) + basic validation
- Vapor API: health, version, auth placeholder
- Postgres + Fluent setup, migrations skeleton
- Zaim integration adapter scaffold (no-op implementation)
- iOS app: modern UI scaffold, API client, SharedKit via SPM (local)
- CI for Swift packages; optional iOS build on macOS runner

### v0.2

- Auth MVP (email/password, JWT, refresh rotation)
- Zaim OAuth connect flow
- Background sync jobs
- Basic charts/visualizations on iOS

### v0.3

- Suggestions (budgets, tagging assists)
- Crypto adapters (read-only integrations)
- Magic link login; optional TOTP MFA
