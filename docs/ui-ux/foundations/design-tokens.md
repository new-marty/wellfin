# Design Tokens (Concise)

## Color
- Palette: primary, success, warning, danger, neutral (0–900)
- Roles: background, surface, text, subtle, focus, border
- Dark mode: tone-preserving; avoid pure black/white

## Typography
- Scales: LargeTitle, Title, Headline, Body, Footnote, Caption
- Line height and truncation rules
- Numeric: tabular for currency in lists and charts

## Spacing & Radius
- Spacing scale: 0, 4, 8, 12, 16, 20, 24, 32
- Radii: 8 (cards), 12 (modals), 4 (chips)

## Elevation
- Levels: none, raised, overlay (blur + dim for modals)

## Motion
- Duration: 120–220ms; respect Reduce Motion
- Easing: standard, decel, accel
